IMPORATANT: Learn how to convert adjacency matrix or edge/vertex into adjacency list and then apply BFS/DFS. Needed for all graph questions - topo sort, UF, shortest path etc. Basically we need Graph inputs as either in ( 2D matrix format ) OR (Adjacency list format ) for solving any graph problems.

 

  • NOTE-1: DFS are recursive calls (with base case) that iterates over - left & right for Trees, 4/8 directions for 2D Grid, all neighbor for Grpahs.
  • NOTE-2: BFS are iterative calls using deque. Also grpahs would need additional ds visited to track visited nodes, unlike Tree
  • NOTE-3: Cyclic Detection in Directed Graphs needs Topological Sort & for undirected graphs needs Union-Find / Discrete Sets.
  • NOTE-4: Shortest Path - Unweighted use BFS. Weighted use Dijkstra (+weights), Bellman Ford (-ve weights)
  • NOTE-5: How to approach general graph problems / thought-process:
    • Undirected and Unweighted graphs ( 2D Grid 200 or graphs-133) use Simple DFS, BFS, UF.
    • Directed (& Unweighted) Graphs - use Topological Sort
    • Weighted Grpahs - Dijkstra or Bellman Ford Algo (Shortest Path) - Both Directed & Undirected || Kruskal / Prims Algo (MST) - Only Undirected.
    • Weighted Grpah Algorithms
